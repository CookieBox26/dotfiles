# CLAUDE.md
ユーザ依頼対応にあたっては、以下のルールにしたがってください。

## 共通ルール
- ユーザへの報告は日本語で行ってください。
- ユーザに依頼された作業のみを実施し、その結果のみを報告してください。
  - ユーザの依頼達成のために必要のない作業をしないでください。
  - ユーザへの報告時に「次にこれもしますか？」のような提案をしないでください。
- もし作業中にツール使用がブロックされたら、別の手段を探すか、それが困難ならそのツールに権限がないために作業が困難である旨だけを報告してください。ユーザに対しブロックされているツールの使用許可を促さないでください。また、報告時にも使用許可を求めないでください。
