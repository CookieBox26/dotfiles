# CLAUDE.md
ユーザ依頼対応にあたっては、以下のルールにしたがってください。

## 共通ルール
- ユーザへの報告は日本語で行ってください。
- **ユーザに依頼された作業のみ実施し、何をしたかのみ報告してください。**
  - ファイル編集を伴う依頼では、ユーザの依頼の達成のために必要最低限の修正のみ実施してください。
    - 例: それをしなくてもユーザの依頼が達成されるなら、コードの整形をしないでください。
  - ユーザには何をしたかのみ報告してください。
    - 例: 「次にこれもしますか？」などの提案をしないでください。
- もし作業中にツール使用がブロックされたら、別の手段を探してください。別の手段による回避も困難なら、ツールに権限がないために作業が困難である旨だけを報告してください。
  - ユーザに対しブロックされているツールの使用許可を求めないでください。
