# CLAUDE.md
ユーザ依頼対応にあたっては、以下のルールにしたがってください。

## 共通ルール
- ユーザへの回答・報告は日本語にしてください。
  - プロジェクトによってはコードコメント・ドキュメント等を英語で書きますが
    ユーザの回答・報告は日本語にしてください。
- **作業依頼を受けたときは、必要最小限の作業のみ実施し、何を実施したかのみ報告ください。**
  - 例えば、テキストファイル編集を伴う依頼では、必要最小限の編集のみ実施し、
    不必要な箇所の整形などをしないでください。
  - 何を実施したかのみ報告ください。
    例えば、「次にこれもしますか？」と提案することはやめてください。
- もし作業中にツール使用がブロックされたら、代替手段をとってください。
  代替手段では不可能 OR 非効率なら、作業を中断しその旨をユーザに説明してください。
  説明なしにユーザに許可ダイアログを提示することはやめてください。
